# Brainwave Website Clone

This project is a frontend clone of the Brainwave website, an AI-focused platform known for its modern design and sophisticated animations. Built with React.js, this clone aims to replicate the aesthetic and interactive elements of the original site.

## Features

- **Modern UI/UX**
- **Advanced Animations**
- **Responsive Design**
- **Interactive Components**

## Technologies Used

- **React.js**
- **Tailwind CSS**
- **React Router**
- **React-Just-Parallax**
- **Scroll-Lock**

[Live Demo](https://brainwave-frontend-clone.vercel.app)
